Política de Privacidade – AppsRiad

Última atualização: Julho de 2025

1. Coleta de dados

O AppsRiad pode coletar áudio de voz para processar comandos de voz. Importante: nenhum áudio ou metadado de voz é armazenado de forma permanente no dispositivo ou em servidores.

2. Uso de dados

Os dados de voz capturados são usados apenas para:

  • Interpretar e executar os comandos solicitados pelo usuário;
  • Melhorar o reconhecimento de voz local (offline).

3. Compartilhamento de dados

Não compartilhamos quaisquer dados pessoais ou de voz com terceiros, exceto:

  • Provedores de anúncios (conforme seção 4), que recebem apenas dados não-identificáveis.

4. Anúncios

O app utiliza o Google AdMob para exibir propagandas.

  • Não coletamos informações pessoais para segmentação;
  • Dados anônimos de uso (sem identificação do usuário) podem ser enviados ao AdMob para melhorar a pertinência dos anúncios.

5. Permissões solicitadas

  • Microfone (android.permission.RECORD_AUDIO): necessário para captar os comandos de voz.
  • Internet (android.permission.INTERNET): necessária para exibição de anúncios e eventuais verificações de atualização.

6. Segurança dos dados

Todo processamento de voz ocorre localmente no dispositivo. Nenhum dado de voz é enviado ou armazenado em servidores próprios.

7. Direitos do usuário

O usuário tem direito de:

  • Solicitar a exclusão de quaisquer dados de voz residuais no dispositivo;
  • Esclarecer dúvidas sobre a política de privacidade.

Para isso, entre em contato via e-mail: contato@appsriad.com.br
