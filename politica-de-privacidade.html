Política de Privacidade

Transparência e segurança avançada no Lista Voxx PRO

Vigente a partir de: 24 de Dezembro de 2025

1. Coleta de Dados

O aplicativo Lista Voxx PRO coleta as seguintes informações:

  • Informações fornecidas pelo usuário ao criar listas de compras ou comandos de voz.
  • Informações anônimas de uso do aplicativo, para fins de análise e melhorias técnicas.
  • Identificador de publicidade do Google (quando o usuário consente explicitamente).

2. Uso de Dados

As informações coletadas são usadas exclusivamente para:

  • Permitir o funcionamento das funções inteligentes e personalização do Lista Voxx PRO.
  • Facilitar o compartilhamento de listas via links externos por decisão do usuário.
  • Exibir anúncios relevantes através de parceiros (aplicável à versão suportada por anúncios).

3. Compartilhamento

O Lista Voxx PRO não compartilha nem vende dados pessoais a terceiros. As listas são de uso pessoal e privado. O compartilhamento externo ocorre apenas por ação voluntária do usuário (ex: enviar para WhatsApp ou E-mail).

4. Publicidade

O Lista Voxx PRO utiliza o Google AdMob. O usuário possui total controle sobre seu ID de publicidade, podendo redefini-lo ou desativá-lo nas configurações de privacidade do sistema Android.

5. Interações entre Usuários

O Lista Voxx PRO não possui rede social própria ou chat integrado. Não há publicação de conteúdo gerado pelo usuário para audiências públicas dentro da plataforma.

6. Segurança

Proteção de Dados no Lista Voxx PRO Priorizamos o armazenamento local. O aplicativo não solicita e não armazena dados sensíveis como CPF, endereço físico ou informações bancárias.

7. Crianças e Adolescentes

O Lista Voxx PRO não é direcionado para menores de 16 anos. Não realizamos a coleta intencional de dados de crianças.

8. Alterações

Esta política pode ser atualizada periodicamente para refletir melhorias no Lista Voxx PRO. Recomendamos a consulta regular a esta página.
