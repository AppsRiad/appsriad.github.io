Política de Privacidade

Sua privacidade é nossa prioridade no Lista Voxx

Última atualização: 24 de dezembro de 2025

1. Coleta de Dados

O aplicativo coleta as seguintes informações:

  • Informações fornecidas pelo usuário ao criar listas de compras ou comandos de voz.
  • Informações anônimas de uso do aplicativo, para fins de análise e melhorias.
  • Identificador de publicidade do Google (quando o usuário consente).

2. Uso de Dados

As informações coletadas são usadas para:

  • Permitir o funcionamento do app e a personalização da experiência do usuário.
  • Compartilhamento de listas criadas pelo usuário com outros contatos por meio de links ou aplicativos de mensagem.
  • Exibir anúncios relevantes (se o usuário estiver usando a versão gratuita com anúncios).

3. Compartilhamento

O app não compartilha dados pessoais com terceiros. As listas criadas pelo usuário são de uso pessoal e, se compartilhadas, o são apenas por ação do próprio usuário por meio de outros aplicativos (como WhatsApp ou e-mail).

4. Publicidade

O app pode usar serviços como o Google AdMob. O usuário pode redefinir ou desativar o ID de publicidade nas configurações do dispositivo Android.

5. Interações entre Usuários

O compartilhamento de listas é voluntário. Não há rede social, chat integrado ou interação pública entre usuários dentro do ambiente do app.

6. Segurança

Segurança Máxima: Todos os dados são armazenados localmente ou anonimizados. O app não solicita informações sensíveis (CPF, dados bancários, etc).

7. Crianças e Adolescentes

Aviso: Este aplicativo não é direcionado para menores de 16 anos. Não coletamos intencionalmente informações de crianças.
