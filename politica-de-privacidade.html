Política de Privacidade – Lista Voxx

Esta Política de Privacidade descreve como o aplicativo Lista Voxx, desenvolvido por Apps Riad, coleta, utiliza e protege as informações fornecidas pelos usuários.

1. Público-alvo

Este aplicativo está disponível para usuários a partir dos 13 anos de idade, incluindo adolescentes e adultos. O aplicativo não é direcionado a crianças menores de 13 anos.

2. Dados Coletados

O app pode coletar automaticamente as seguintes informações:

  • ID de publicidade do Google (Advertising ID / AD_ID)
  • Dados de uso anônimos (tempo de uso, cliques, interações)
  • Informações de dispositivo não pessoais (modelo, idioma, sistema operacional)

3. Uso das Informações

As informações coletadas são utilizadas para:

  • Exibição de anúncios via Google AdMob
  • Melhoria da experiência do usuário
  • Análise de métricas de uso do app

4. Anúncios e Terceiros

Este app utiliza o Google AdMob para exibir anúncios. O AdMob pode utilizar o AD_ID para personalizar os anúncios com base no perfil do usuário, de acordo com as configurações do dispositivo.

Para usuários com idade entre 13 e 17 anos, os anúncios exibidos são adequados para essa faixa etária.

5. Consentimento

Ao utilizar o app, o usuário concorda com a coleta e uso dos dados descritos nesta política. É possível desativar a personalização de anúncios nas configurações do dispositivo Android.

6. Compartilhamento de Dados

O app não compartilha seus dados com terceiros, exceto com os serviços necessários para funcionamento, como o Google AdMob.

7. Segurança

Os dados coletados são tratados com medidas razoáveis de segurança e utilizados apenas para as finalidades descritas.

8. Alterações nesta Política

Esta política pode ser atualizada periodicamente. Recomendamos que os usuários consultem esta página regularmente.

9. Contato

Para dúvidas sobre esta Política de Privacidade, entre em contato pelo e-mail: appsriad@gmail.com
